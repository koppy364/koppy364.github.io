# 1. React
html, css, javascript 도와주는 도구 (라이브러리) 가장 많이사용한다.


1. single page application 만들수있음
2. component로 html 재사용 편리
3. 데이터가 html에 자동 반영됨
4. react native로 앱개발 가능
